Orientações de uso para as pastas pré existentes

- external
  * libs que não estão disponível no npm, não não funcionam com o commonJS ou tem uso muito especifico não pertencendo 
  a nenhum modulo

- js 
  * todos os modulos da aplicação oquestrado pelo browserify

- resources
  * css, imagnes, entre outros

- views
  * html ou arquivos baseados em algum template engine, os arquivos seguem a mesma divisão de modulos do .js
    usando usando o conceito de partials para angular.
